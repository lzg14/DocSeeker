# Changelog

## [Unreleased]

### feat

- **OCR 进度反馈**：右键菜单 OCR 扫描功能新增实时进度显示，识别每张图片时显示"正在识别第 X 张图片... (X/N)"及进度条，完成后自动更新搜索索引
- **图片 OCR 识别**：新增图片文字识别功能。启用后可从图片（PNG/JPG/BMP/TIFF 等）中提取文字内容，使图片中的文字也可被搜索。采用 Windows.Media.Ocr 引擎，支持中文、英文、日文、韩文等语言。
- **文献引用卡片**：新增从文档中提取关键词引用的功能。点击文件后可提取包含关键词的引用（前后至少2个完整句子，各100字以上），支持复制全部、导出为 MD/TXT 格式。支持 PDF、Word、Excel、PPT、TXT、MD 等77种文档格式。

### fix

- 修复文献引用提取功能在关键词靠近文档开头或结尾时返回0条结果的问题
