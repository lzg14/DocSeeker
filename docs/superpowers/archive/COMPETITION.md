# DocSeeker 竞品分析报告

> 更新时间: 2026-04-23

## 一、项目定位

**DocSeeker** 是一款基于 Electron + React + TypeScript 的本地文档全文搜索工具，面向需要管理大量个人积累文档的用户，提供离线优先的隐私保护搜索体验。

**核心技术栈：**
- 全文索引：SQLite FTS5 + BM25 相关性排名
- 内容解析：mammoth (DOCX)、xlsx (Excel)、jszip (PPTX)、pdf-parse (PDF)
- 图片 OCR：Tesseract.js
- 邮件解析：pst-extractor (PST/Outlook)
- 桌面框架：Electron + React 18 + TypeScript
- 数据库：better-sqlite3（原生模块）

## 四、DocSeeker 核心优势

1. **FTS5 + BM25 专业级排序** - 相关性排序优于竞品
2. **唯一的中英双语原生支持** - 竞品均无国际化
3. **77 种格式支持，已全部验证** - 最全面的格式支持
4. **模糊搜索 (Fuse.js)** - 竞品唯一
5. **图片 OCR 文字识别** - 仅 Copernic 付费版有
6. **Outlook PST/MSG 邮件支持** - 解析大型邮件归档
7. **词干智能搜索** - 英文词干提取
8. **NTFS USN Journal 实时监控** - 与 Everything 相同技术
9. **纯本地隐私** - 无云端上传
10. **7 种主题 + 国际化** - 最优用户体验

## 五、DocSeeker 核心不足

| 不足 | 影响 | 竞品对比 |
|------|------|---------|
| 无正则表达式搜索 | 高级用户无法精准查询 | Everything/AnyTXT 均支持 |
| 无网络驱动器支持 | 无法索引 NAS | AnyTXT/Copernic 有 |
| 仅 Windows | 跨平台未实现 | DocFetcher/Recoll 全平台 |
| Electron 冷启动慢 | 响应不如 Everything | Everything 毫秒级响应 |
